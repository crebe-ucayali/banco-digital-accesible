Biblioteca Accesible

Espacio organizado para reunir libros digitales, bases académicas, repositorios abiertos y accesos de consulta para docentes, estudiantes y familias.

Biblioteca digital

Accede a recursos de consulta

La biblioteca se organiza como un submódulo del Banco Digital Accesible. Su función es ordenar los accesos a materiales de lectura, fuentes académicas y recursos externos autorizados, sin mezclar su finalidad con los espacios de LSP o Braille.

Ruta recomendada de uso

Primero se consulta el catálogo institucional. Luego, si el material requerido no se encuentra disponible, se revisan fuentes abiertas, bases académicas o repositorios externos. Esta organización ayuda a evitar enlaces dispersos y facilita el mantenimiento del módulo.

Ver catálogo Ver organización

Criterio de incorporación

Los materiales deben ser propios, autorizados, de dominio público o enlazados desde fuentes oficiales. Las licencias institucionales se muestran como orientación de acceso, pero las claves o credenciales no deben publicarse de forma abierta.

Organización sugerida

Modelo de crecimiento del módulo

La mejor alternativa es mantener esta biblioteca como una sección propia dentro de BDA, con enlaces controlados y categorías estables. Si más adelante aumenta el número de recursos, se puede migrar el catálogo a una hoja de cálculo, una base de datos o un servicio externo.

1. Catálogo básico

Registro manual de recursos con título, tipo, descripción, público recomendado y enlace de acceso.

2. Almacenamiento externo

Uso de carpetas institucionales o servicios de almacenamiento para evitar cargar archivos pesados dentro del repositorio.

3. Accesos protegidos

Orientación pública para el usuario y entrega privada de credenciales cuando existan licencias institucionales.

4. Actualización periódica

Revisión de enlaces, retiro de recursos vencidos e incorporación de nuevas fuentes según necesidad educativa.

Esta biblioteca no debe publicar libros protegidos sin autorización. Lo más seguro es enlazar fuentes oficiales, materiales propios, documentos autorizados o recursos con licencia abierta.
