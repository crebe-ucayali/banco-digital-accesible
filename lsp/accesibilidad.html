Criterios de uso accesible del banco visual de Lengua de Señas Peruana para una consulta clara, ordenada y funcional.

Criterios de uso accesible

Contenido visual del recurso

Este apartado organiza los criterios de accesibilidad del módulo LSP en una lectura similar a preguntas frecuentes, para que cada punto pueda revisarse de manera clara e independiente.

¿Cómo se presenta la consulta visual?

Las señas se muestran mediante imágenes organizadas por categorías, lo que permite ubicar vocabulario de manera rápida y visual dentro del módulo.

¿Cómo ayuda la búsqueda por palabra?

El buscador permite ingresar palabras clave para acceder directamente al vocabulario requerido, sin necesidad de recorrer todo el banco de señas.

¿Para qué sirven las categorías?

Las categorías agrupan las señas por campos de uso, como familia, colores, números, lugares u otros grupos de vocabulario, facilitando una exploración más ordenada.

¿El diseño se adapta a distintos dispositivos?

Sí. La página está preparada para visualizarse en computadora, tablet y celular, procurando que la consulta pueda realizarse desde diferentes pantallas.

¿Se puede navegar con teclado?

Los enlaces, botones, campos de búsqueda y filtros están preparados para ser utilizados mediante teclado, favoreciendo una navegación más accesible.

¿Cómo se cuida el contraste visual?

Se emplean colores con contraste suficiente entre fondo, texto y botones, con el fin de mejorar la lectura y la identificación de los elementos principales.

La plataforma puede seguir ampliándose con videos demostrativos, descripciones complementarias, variantes regionales, ejemplos de uso y recursos descargables para docentes y familias.
