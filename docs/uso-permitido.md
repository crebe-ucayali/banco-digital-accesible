# Uso permitido

Este documento orienta el uso permitido del Banco Digital Accesible.

Banco Digital Accesible es un proyecto educativo digital personal desarrollado y organizado por Gabriel Berrospi. Tiene una finalidad educativa, accesible y no comercial.

## Usos permitidos

El proyecto puede utilizarse para:

- Consulta y organización de recursos accesibles.
- Sensibilización sobre inclusión educativa.
- Apoyo a docentes, familias y profesionales de educación especial.
- Enlace o integración de bancos específicos, como LSP, Braille, materiales educativos, juegos y recursos formativos.
- Presentación de recursos pedagógicos complementarios.
- Adaptaciones del código, siempre que se respeten los créditos y la licencia.
- Uso como referencia para proyectos educativos similares.

## Condiciones de uso

Toda reutilización debe conservar:

- El nombre del proyecto.
- La autoría de Gabriel Berrospi.
- Los créditos de recursos externos utilizados.
- La diferenciación de cambios cuando se realicen adaptaciones.

## Recursos externos

El código del repositorio se rige por el archivo `LICENSE`.

Los materiales externos, imágenes, documentos, recursos del Ministerio de Educación del Perú, logos, videos, fichas, infografías o contenidos de terceros mantienen sus propias condiciones de autoría, uso y difusión.

## Límites de uso

No se debe presentar el proyecto como propio, eliminar créditos, usar identificaciones de terceros sin autorización o publicar versiones modificadas sin identificar claramente los cambios realizados.

El Banco Digital Accesible debe entenderse como plataforma educativa de apoyo para organizar y facilitar el acceso a recursos accesibles. No sustituye la planificación docente, la evaluación profesional ni la intervención especializada.
