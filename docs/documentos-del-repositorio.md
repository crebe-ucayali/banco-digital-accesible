# Documentos del repositorio

Este archivo lista los documentos de respaldo del Banco Digital Accesible.

- `sustento-del-proyecto-personal.md`: documento central del proyecto personal.
- `autoria-y-contexto.md`: autoría, contexto de uso y reconocimiento.
- `aviso-legal.md`: alcance general y uso responsable.
- `fuentes-y-creditos.md`: fuentes, créditos y recursos externos.
- `uso-permitido.md`: usos permitidos y condiciones generales.
- `alcance-pedagogico.md`: alcance pedagógico del banco digital.
- `criterios-de-integracion.md`: criterios para integrar recursos.
- `mapa-de-repositorios.md`: organización de módulos y repositorios relacionados.
- `estructura-del-proyecto.md`: organización de archivos y carpetas.
- `bitacora-de-cambios.md`: registro de avances y mejoras.

Estos documentos ayudan a conservar el orden, la finalidad educativa, la autoría y la trazabilidad del proyecto.
